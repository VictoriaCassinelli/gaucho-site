# Restaurante GAUCHO 🍽️

Este es un sitio web gastronómico ficticio desarrollado para el restaurante argentino **GAUCHO**, usando HTML, CSS, JavaScript, SASS y Bootstrap.

## 📁 Estructura del sitio

```
/gaucho
├── index.html        # Página principal
├── menu.html         # Menú con imágenes y recetas
├── reserva.html      # Formulario de reservas funcional
├── nosotros.html     # Información del equipo
├── admin.html        # Vista interna para consultar reservas
├── css/
│   ├── style.css
│   └── style.scss
├── js/
│   ├── script.js
│   └── admin.js
├── img/
│   └── (instrucciones para añadir imágenes)
```

## 🚀 Funcionalidades destacadas

- Navegación entre páginas HTML
- Formulario de reservas con validación y almacenamiento en localStorage
- Página de administración de reservas
- Menú interactivo con galería y modales con recetas
- Estilo visual profesional usando Bootstrap 5 y SASS

## 🖼️ Imágenes

Las imágenes se cargan desde URLs externas de Pixabay o pueden colocarse manualmente en la carpeta `/img/`.

## ✅ Requisitos

- Navegador moderno (Chrome, Firefox, Edge)
- Editor de código (Visual Studio Code recomendado)

## 📦 Cómo usar

1. Descargar y descomprimir el archivo `.zip`.
2. Abrir el proyecto en tu editor de código.
3. Abrir `index.html` en tu navegador.
4. Subí los archivos a GitHub para publicarlo con GitHub Pages (opcional).
